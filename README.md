# ML Test Práctico - Frontend

La aplicación consta de tres componentes principales: Caja de búsqueda, visualización de resultados, y la descripción del detalle de un producto.

###### Stack tecnológico
 - Cliente:  
    - HTML
    - React
    - CSS

- Servidor:
    - Node v12.6.0 
    - Express ^4.17.1

###### Ejecutar

npm install
npm start
